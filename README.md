## Описание проекта

Многостраничный статический сайт компании NeuralHub, специализирующейся на искусственном интеллекте и машинном обучении. Реализован на чистом HTML5 и CSS3 без использования фреймворков.

## Технологии

- HTML5 (семантическая разметка)
- CSS3 (Flexbox, Grid, Media Queries, Animations, Custom Properties)
- SVG-графика
- Google Fonts (Inter)

## Реализованные лабораторные работы

1. **Структура документа** — семантическая HTML-структура
2. **Основы CSS** — переменные, шрифты, цвета
3. **Блочная модель** — box-sizing, padding, margin, контейнеры
4. **Работа с изображениями** — SVG-иконки, alt-тексты
5. **Flexbox** — навигация, карточки, footer, sr-only
6. **Grid** — Мондриан, dashboard, RAM-галерея (grid.html)
7. **RWD** — mobile-first, media queries, hamburger-меню
8. **Формы** — регистрация, fieldset, label, mailto/tel (form.html)
9. **Псевдо и анимации** — hover, focus, ::before, flip-card, marquee (demo.html)
10. **Доступность** — skip-link, aria, sr-only, focus-visible, lang, landmarks
